# Aplikasi HPP (PHP + SQLite)

Simple aplikasi native PHP untuk menghitung HPP (Harga Pokok Penjualan).

Fitur:
- Landing page
- Register / Login (SQLite)
- Dashboard dengan riwayat perhitungan
- Kalkulator HPP: Persediaan Awal + Pembelian - Persediaan Akhir

Instalasi dan menjalankan (menggunakan built-in PHP server):

1. Pastikan PHP terpasang.
2. Jalankan server di folder projek ini:

```bash
php -S localhost:8000
```

3. Buka `http://localhost:8000` di browser.

Catatan:
- Database SQLite dibuat otomatis pada `data/hpp.db`.
- Akun default: `admin` / `admin`.
